Quick Tips

  • Linear: changes at a constant rate.
  • Slope (m): “rise over run”.
  • y-intercept (b): where the line crosses the y-axis.
  • Function: input → output (x → y).

Lessons

Understand what linear equations are and how linear functions describe relationships.

1) What is a linear equation?

A linear equation is an equation that can be written in the form: Ax + B = C (or with y, like Ax + By = C).

Example: 2x + 3 = 11. To solve it, you do the same thing to both sides until x is alone.

Goal: Get the variable by itself.

2) Solving a linear equation (steps)

  1. Simplify both sides (combine like terms).
  2. Undo addition/subtraction first.
  3. Undo multiplication/division next.
  4. Check your solution by plugging it back in.
Pro tip: Keep equations balanced—whatever you do to one side, do to the other.

3) What is slope (m)?

The slope tells how much the line rises or falls when you move right. Using two points, slope is:

(rise/run) = (y₂ − y₁) / (x₂ − x₁)

If slope is positive, the line goes up. If negative, it goes down.

4) Linear functions (y = mx + b)

A linear function is usually written as: y = mx + b

  • m = slope (rate of change)
  • b = y-intercept (where x = 0)
  • x = input, y = output
When x increases, y changes by a constant amount.

5) Convert words to equations

Many problems describe a linear relationship using words. Here’s a common pattern:

  • “For every 1 unit increase in x…” → slope tells the “change in y”.
  • “When x = 0…” → gives the y-intercept b.
